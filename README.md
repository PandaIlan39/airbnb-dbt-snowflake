# 🏠 Airbnb Analytics Pipeline | dbt + Snowflake

A complete end-to-end data engineering portfolio project built with dbt and Snowflake,
transforming raw Airbnb listings data into clean, analytics-ready models using the
Medallion Architecture (Bronze → Silver → Gold).

---

## 🏗️ Architecture
```
Raw CSV (Airbnb) ──→ Snowflake RAW Schema ──→ Bronze ──→ Silver ──→ Gold ──→ Analytics
Countries Seed   ──→ Snowflake RAW Schema ──────────────────────→ Gold ──→ Analytics
```

### Medallion Architecture
| Layer | Model | Description |
|-------|-------|-------------|
| 🥉 Bronze | `bronze_listings` | Raw data copy with load timestamp. No transformations. |
| 🥈 Silver | `silver_listings` | Cleaned, deduplicated, incremental loading by date |
| 🥇 Gold | `pricing_analysis` | Average prices per neighbourhood and room type |
| 🥇 Gold | `host_performance` | Host rankings by listings and reviews |
| 🥇 Gold | `combined_analysis` | Listings enriched with country reference data |

---

## ⚡ Key Features

- **Medallion Architecture** — Bronze, Silver, Gold layers following industry best practices
- **Incremental Loading** — Silver layer only processes new data based on load_date
- **Data Deduplication** — QUALIFY ROW_NUMBER() removes duplicate listing IDs
- **Audit Trail** — loaded_at timestamp and load_date on every record
- **Data Quality Tests** — 16 automated tests across all models
- **Auto Documentation** — full lineage diagram generated by dbt docs

## 🚀 How to Run This Project

### Prerequisites
- A free Snowflake account — sign up at snowflake.com
- Python 3.11 — download at python.org (do NOT use Python 3.12 or higher)
- dbt-snowflake installed

---

### Step 1: Clone the Repository
```bash
git clone https://github.com/PandaIlan39/airbnb-dbt-snowflake.git
cd airbnb-dbt-snowflake
```

---

### Step 2: Install dbt
```bash
pip install dbt-snowflake
```
Verify it installed:
```bash
dbt --version
```

---

### Step 3: Set Up Snowflake
Log into Snowflake and run:
```sql
USE ROLE ACCOUNTADMIN;

CREATE WAREHOUSE IF NOT EXISTS AIRBNB_WH
  WAREHOUSE_SIZE = 'X-SMALL'
  AUTO_SUSPEND = 60
  AUTO_RESUME = TRUE;

CREATE DATABASE IF NOT EXISTS AIRBNB_DB;
CREATE SCHEMA IF NOT EXISTS AIRBNB_DB.RAW;

USE WAREHOUSE AIRBNB_WH;
USE DATABASE AIRBNB_DB;
USE SCHEMA RAW;
```

---

### Step 4: Load the Airbnb Data
1. Download the Airbnb NYC dataset from kaggle.com
2. In Snowflake UI go to Data → Add Data → Load files into a Stage
3. Create a stage called `AIRBNB_STAGE` and upload the CSV
4. Run this to create the table and load data:
```sql
CREATE OR REPLACE TABLE AIRBNB_DB.RAW.LISTINGS (
    id NUMBER, NAME VARCHAR, host_id NUMBER,
    host_identity_verified VARCHAR, host_name VARCHAR,
    neighbourhood_group VARCHAR, neighbourhood VARCHAR,
    lat FLOAT, long FLOAT, country VARCHAR,
    country_code VARCHAR, instant_bookable VARCHAR,
    cancellation_policy VARCHAR, room_type VARCHAR,
    construction_year NUMBER, price VARCHAR,
    service_fee VARCHAR, minimum_nights NUMBER,
    number_of_reviews NUMBER, last_review DATE,
    reviews_per_month FLOAT, review_rate_number NUMBER,
    calculated_host_listings_count NUMBER,
    availability_365 NUMBER, house_rules VARCHAR,
    license VARCHAR
);

COPY INTO AIRBNB_DB.RAW.LISTINGS
FROM @AIRBNB_STAGE
FILE_FORMAT = (
    TYPE = 'CSV'
    FIELD_OPTIONALLY_ENCLOSED_BY = '"'
    PARSE_HEADER = TRUE
    NULL_IF = ('', 'NULL')
    EMPTY_FIELD_AS_NULL = TRUE
)
MATCH_BY_COLUMN_NAME = CASE_INSENSITIVE
ON_ERROR = 'CONTINUE';
```

---

### Step 5: Configure dbt Connection
```bash
dbt init airbnb_project
```
Fill in your Snowflake details:
- account → found in your URL before .snowflakecomputing.com
- user → your Snowflake username
- password → your Snowflake password
- role → ACCOUNTADMIN
- warehouse → AIRBNB_WH
- database → AIRBNB_DB
- schema → RAW
- threads → 1

Test the connection:
```bash
dbt debug
```
You should see: `All checks passed!`

---

### Step 6: Run the Full Pipeline
```bash
# Load reference data
dbt seed

# First run - full refresh to load all data
dbt run --full-refresh

# After first run - incremental loads only process new data
dbt run

# Run all data quality tests
dbt test

# View documentation and lineage diagram
dbt docs generate
dbt docs serve
```
Then open your browser at `http://localhost:8080`

---

### Step 7: Verify Results in Snowflake
```sql
-- Check row counts
SELECT COUNT(*) FROM AIRBNB_DB.RAW.BRONZE_LISTINGS;
SELECT COUNT(*) FROM AIRBNB_DB.RAW.SILVER_LISTINGS;

-- Check incremental load audit trail
SELECT load_date, COUNT(*) AS rows_loaded
FROM AIRBNB_DB.RAW.SILVER_LISTINGS
GROUP BY load_date
ORDER BY load_date DESC;

-- Check gold layer
SELECT * FROM AIRBNB_DB.RAW.PRICING_ANALYSIS ORDER BY avg_price DESC;
SELECT * FROM AIRBNB_DB.RAW.HOST_PERFORMANCE ORDER BY total_listings DESC;
SELECT * FROM AIRBNB_DB.RAW.COMBINED_ANALYSIS LIMIT 10;
```
